## Deploy

1. Push project to GitHub.
2. Create a Vercel account.
3. Import the GitHub repo into Vercel.
4. Deploy.
5. Add custom domain:

spartyncapital.com

6. In domain registrar DNS, follow Vercel's DNS instructions.
